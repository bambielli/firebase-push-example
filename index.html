Explanation:

This form uses Firebase to persist data, and is a proof of concept for using the `push` method for storing data.

The delete buttons trigger a `set` operation at the key of the row to be deleted, which is achieved by setting the key to null.

Firebase's "Realtime Database" is particularly interesting: value changes are pushed to all connected clients within miliseconds!

Try it out by opening this page in several windows, and adding / removing values in each.
